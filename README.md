# 🌱 Farmer Crop Investment System with Weather Forecast & Generative AI

## 📌 Project Overview
This is a Full Stack Internship Project developed to help farmers manage crop investments, expenses, weather forecasting, and profit/loss analysis digitally.

The application provides real-time features for:
- Farmer Management
- Crop Management
- Investment Tracking
- Weather Forecast
- Dashboard & Reports
- AI-based Insights
